Text-Saver

This is a text saving and retrieval web app built in Django. This app is able to save short text snippets with a title, timestamp and created user. The snippet can also have a tag. Implemented JWT authentication for getting user.

API Details

Authentication API:

Login API

POST /api/login

Refresh API

POST /api/refresh

CRUD APIs:

Overview API

GET /api/snippets-overview

Create API

(Need to Authenticate)

POST /api/snippets-create

Detail API

GET /api/snippets-details/<snippet_id>

Update API

PUT /api/snippets-update/<snippet_id>

Delete API

DELETE /api/snippets-delete/<snippet_id>

Tag List API

GET /api/tag-list/

Tag Detail API

GET /api/tag-details/<tag_id>
